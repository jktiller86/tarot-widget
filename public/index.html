How to Embed

Add this code to your website:

<div id="tarot-widget-root"></div>
<script src="https://tarot-widget.netlify.app/tarot-widget.js"></script>
<script>
  TarotCardWidget.init({
    root: '#tarot-widget-root',
    subscribeEndpoint: 'https://tarot-widget.netlify.app/.netlify/functions/subscribe'
  });
</script>
